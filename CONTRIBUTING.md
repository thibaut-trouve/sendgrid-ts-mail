## Instructions

These steps will guide you through contributing to this project:

- Fork the repo
- Clone it and install dependencies

		git clone https://github.com/thibaut-trouve/sendgrid-ts-mail
		npm install
    develop

Finally send a [GitHub Pull Request](https://github.com/thibaut-trouve/sendgrid-ts-mail/compare?expand=1) with a clear list of what you've done (read more [about pull requests](https://help.github.com/articles/about-pull-requests/)). Make sure all of your commits are atomic (one feature per commit).
